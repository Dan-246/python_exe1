ウォーミングアップ1：計算の表示

目標時間：5分

目標: Pythonで簡単な計算を行い、その結果を直接ターミナルに表示しよう。

Pythonを使って、以下の計算結果をターミナルに表示するプログラムを書いてみましょう。

  1. 1
    `2の15乗` の答えを表示してください。
  2. 2
    `123456 ÷ 7` の**商**（割り算の結果の整数部分）を表示してください。
  3. 3
    `123456 ÷ 7` の**余り**を表示してください。
コード例:

# 123456 ÷ 7 の商
print(123456 // 7)

# 123456 ÷ 7 の余り
print(123456 % 7)

    問題3：学力判定チェック

    目標時間：15分

    目標: 比較演算子と論理演算子（`and`, `or`）を使って、いくつかの条件が正しいかどうか（`True` か `False`）を確認して表示しよう。

    テストの点数から、学力の状況をチェックするプログラムです。特定の条件が満たされているかどうかを `True`（正しい）か `False`（間違っている）で確認してみましょう。

    1. 1
      以下の情報を変数に代入してください。（あなたの好きな点数を入れてみよう）
      • `japanese_score` という変数に国語の点数（整数）を代入。
      • `math_score` という変数に算数の点数（整数）を代入。
      • `science_score` という変数に理科の点数（整数）を代入。
    2. 2
      比較演算子（`>`, `<`, `==`, `>=`, `<=`）と論理演算子（`and`, `or`）を使って、以下の条件が正しいかどうかを `print()` で直接ターミナルに表示してみましょう。（`True` か `False` が表示されます。）
      • 判定A: 国語の点数が80点以上 かつ 算数の点数が70点以上である。
      • 判定B: 理科の点数が60点未満 または 国語の点数が50点未満である。
      • 判定C: 算数の点数が100点である。
      • 判定D: （応用）理科の点数が70点以上 かつ （国語の点数が80点以上 または 算数の点数が80点以上）である。

    ヒント: `print()` の中に直接条件式を書いてみよう！例えば、`print(10 > 5)` と書くと `True` が表示されるよ。

    コード例: 
    japanese_score = 85
math_score = 72
science_score = 65

# 判定A
print("判定A: 国語の点数が80点以上 かつ 算数の点数が70点以上であるか？")
print(japanese_score >= 80 and math_score >= 70)

# 判定B
print("\n判定B: 理科の点数が60点未満 または 国語の点数が50点未満であるか？")
print(science_score < 60 or japanese_score < 50)

# 判定C
print("\n判定C: 算数の点数が100点であるか？")
print(math_score == 100)

# 判定D (応用)
print("\n判定D: 理科の点数が70点以上 かつ (国語の点数が80点以上 または 算数の点数が80点以上)であるか？")
print(science_score >= 70 and (japanese_score >= 80 or math_score >= 80))

    問題5：誕生日、何曜日？

    目標時間：10分

    目標: 「ツェラーの公式」を使って、自分の誕生日が何曜日だったかを計算してみよう！`input()`関数を使って、自分で日付を入力するよ。

    皆さんの誕生日が何曜日だったか、気になりませんか？「ツェラーの公式」という数学の公式を使うと、どんな日付でも曜日を計算することができます。

    この問題では、あなたが生まれた年、月、日をプログラムに入力し、ツェラーの公式を使って曜日を計算してみましょう！

    ツェラーの公式について

    ツェラーの公式は、日付の曜日を求めるための数学の公式です。見た目は複雑ですが、一つずつ計算すれば大丈夫！

    ※ 上の画像はツェラーの公式を表しています。 この演習では、画像に示されている公式の記号を使って計算します。 （MathJaxは埋め込んでいますが、この公式の表示には画像を使用しています。）

    なお、記号の意味は以下の通りです。

    • h
      計算結果が示す曜日です。この数字が何曜日になるか、下のリストで確認しましょう。
      • 0: 土曜日
      • 1: 日曜日
      • 2: 月曜日
      • 3: 火曜日
      • 4: 水曜日
      • 5: 木曜日
      • 6: 金曜日
    • d
      計算したい日付の**「日」**の数字です。（例：1日なら `1`、25日なら `25`）
    • m
      計算したい日付の**「月」**の数字です。ただし、特別なルールがあります！
      • 3月から12月: そのままの数字を使います。（例: 3月は `3`、12月は `12`）
      • 1月と2月: それぞれ `13` と `14` として扱います。このとき、**年を1年前にして**計算します。（例: 2000年1月なら、年を1999年、月を13として計算します）
    • Y
      計算したい**「年」の下2桁**の数字です。（例：2023年なら `23`）Pythonでは `year % 100` で求められます。
    • y
      計算したい**「年」全体**の数字です。（例：2023年なら `2023`）

    課題:

    1. 1
      `input()`関数を使って、あなたの**生まれた年（西暦）**、**月**、**日**をそれぞれ入力し、変数に代入してください。プログラムを実行するとターミナルに質問が表示されますので、キーボードで入力してください。`input()`で受け取ったものは**文字列**なので、`int()`を使って**整数**に変換するのを忘れないでください。
    2. 2
      計算に必要な `d`, `m`, `Y`, `y` の各値を、上で説明したルールに従って変数に代入してください。
    3. 3
      ツェラーの公式の計算式をPythonで記述し、結果を `h` という変数に代入してください。
    4. 4
      `h` の値（0〜6）に対応する曜日をターミナルに表示してください。
    表示例（例：2000年1月1日を入力した場合）: 
    生まれた年（西暦）を入力してください: 2000
生まれた月を入力してください: 1
生まれた日を入力してください: 1
あなたの誕生日は土曜日でした！
    コード例: 
    # 1. 生まれた年、月、日を入力
year = int(input("生まれた年（西暦）を入力してください: "))
month = int(input("生まれた月を入力してください: "))
day = int(input("生まれた日を入力してください: "))

# 2. ツェラーの公式に必要な各値を準備
d = day          # 日
m = month        # 月（調整済み）
Y = year % 100   # 年の下2桁
y = year         # 年全体

# 3. ツェラーの公式を計算
# h = (d + 小数点以下切り捨て(26*(m+1)/10) + Y + 小数点以下切り捨て(Y/4) - 2*小数点以下切り捨て(y/100) + 小数点以下切り捨て(y/400)) を7で割った余り
h = (d + (26 * (m + 1)) // 10 + Y + Y // 4 - 2 * (y // 100) + y // 400) % 7

# 4. 結果の表示
days_of_week = ["土曜日", "日曜日", "月曜日", "火曜日", "水曜日", "木曜日", "金曜日"]
print("あなたの誕生日は" + days_of_week[h] + "でした！")
